# Configuration

## Quiz Layout Requirement

The plugin only works when:

Questions per page = 1

If multiple questions per page are selected, monitoring will be unavailable.

---

## Capabilities

The following capabilities are available:

- quizaccess/cheatdetect:view
- quizaccess/cheatdetect:manage
- quizaccess/cheatdetect:viewcoursereports

Manage them via:

Site administration → Users → Permissions → Define roles
