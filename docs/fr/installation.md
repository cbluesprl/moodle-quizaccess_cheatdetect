# Installation

## Installation manuelle

1. Copier le dossier du plugin dans :

   mod/quiz/accessrule/cheatdetect

2. Aller dans :

   Administration du site → Notifications

3. Finaliser l'installation.

---

## Après installation

- Purger les caches.
- Vérifier que la règle d’accès est disponible dans les paramètres du quiz.
