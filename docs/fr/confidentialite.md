# Confidentialité

## Données enregistrées

Le plugin enregistre :

- Identifiant utilisateur
- Identifiant du quiz
- Identifiant de tentative
- Slot de question
- Métriques comportementales
- Événements capturés
- Informations sur extensions détectées

---

## Conformité RGPD

Le plugin implémente l’API Privacy de Moodle.

Il permet :

- L’export des données utilisateur
- La suppression des données utilisateur
